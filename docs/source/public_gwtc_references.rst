Public GWTC cWB References
==========================

The public cWB documentation includes waveform-reconstruction reports and
Coherent Event Display (CED) pages for selected GWTC events. This page keeps
those references discoverable from the pycWB documentation without importing
the full public gallery.

.. note::

   These links point to public cWB/cWB-2G outputs. They are reference material,
   not pycWB tutorials. To run a current pycWB search, start with
   :ref:`start_here` or :ref:`standard_analysis`.

Summary Plot Categories
-----------------------

The public cWB catalog pages organize cWB comparisons with modeled CBC
reconstructions into the same broad plot categories:

* maximum-likelihood whitened waveforms in the time domain;
* whitened waveform residuals;
* skymaps comparing cWB with parameter-estimation results;
* cWB confidence intervals in time, frequency, and time-envelope views;
* detector strain noise amplitude spectral density (ASD).
